# @autotranslate/next

Next.js integration for autotranslate. App Router, Pages Router, RSC, and
middleware-based locale routing.

```bash
pnpm add @autotranslate/next @autotranslate/react @autotranslate/core
```

## Exports

- `@autotranslate/next` — server helpers (`getT`, `getLocale`)
- `@autotranslate/next/plugin` — `withAutotranslate(nextConfig)`
- `@autotranslate/next/middleware` — `createNextMiddleware()` for locale routing
