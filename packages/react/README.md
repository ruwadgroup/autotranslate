# @autotranslate/react

React adapter for
[autotranslate](https://github.com/tamimbinhakim/autotranslate). Hooks,
components, and providers that work in client components, RSC, Vite, Remix, and
React Native.

```bash
pnpm add @autotranslate/react @autotranslate/core
```

## Exports

- `@autotranslate/react` — `<T>`, `<Var>`, `<Plural>`, `<Num>`, `<Currency>`,
  `<DateTime>`, `useT`, `TranslationProvider`
- `@autotranslate/react/server` — `createTranslator`, `getT` (for RSC / SSR)
