# @autotranslate/mcp

[Model Context Protocol](https://modelcontextprotocol.io) server exposing
autotranslate's `extract` / `translate` / `check` operations as tools. Lets
agentic clients (Claude Code, Cursor, …) drive translations end-to-end.

> Status: **planned for v0.6.** The current binary prints a placeholder. See
> [`ROADMAP.md`](../../ROADMAP.md).

```bash
pnpm dlx @autotranslate/mcp
```
